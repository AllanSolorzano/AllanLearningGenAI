# Agent Workspace

This directory is for participant AI SRE agent implementations.

The platform does not require a specific framework. Teams may use LangGraph, CrewAI, AutoGen, Semantic Kernel, PydanticAI, OpenAI SDK, Anthropic SDK, Gemini SDK, MCP servers, or custom Python.

Use [starter-kit](starter-kit/README.md) as a lightweight reference for:

- agent role boundaries
- planning records
- safety guardrails
- tool abstraction interfaces
- RCA handoff shape

The starter kit is intentionally incomplete. It should help teams avoid unsafe patterns without becoming a hardcoded solution to the competition.
